# Íconos de GC Apartamentos

## Ubicación de íconos

### Ícono de la aplicación
- **Archivo**: `icon.png`
- **Ubicación**: `gc_apartamentos/static/description/icon.png`
- **Tamaño recomendado**: 256x256 píxeles
- **Formato**: PNG con fondo transparente o de color

Este ícono se muestra en:
- Lista de aplicaciones de Odoo
- Menú principal de la aplicación

### Instrucciones
1. Coloca tu archivo de imagen `icon.png` en esta carpeta
2. El archivo debe ser PNG de 256x256 píxeles
3. Si no proporcionas un ícono, Odoo usará el ícono predeterminado

### Generación de ícono
Puedes usar herramientas como:
- GIMP
- Photoshop
- Figma
- Canva
- Generadores de íconos online

### Actualización
Después de agregar el ícono:
1. Actualiza el módulo en Odoo
2. Refresca el navegador (Ctrl+F5)
